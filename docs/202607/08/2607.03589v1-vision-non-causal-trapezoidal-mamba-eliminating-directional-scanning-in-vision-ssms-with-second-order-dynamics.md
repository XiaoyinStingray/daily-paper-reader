## 摘要
状态空间模型（SSMs）已成为视觉Transformer的替代方案，但大多数视觉SSM从因果序列建模中继承了方向性令牌扫描。虽然方向扫描对序列数据有效，但它引入了空间偏差和方向敏感表示。我们提出了视觉非因果梯形Mamba（VNCT），这是一种二阶非因果视觉SSM，使所有图像令牌能够在单次传递中交互，消除了方向扫描，并实现了低单图像推理延迟。VNCT展现出更鲁棒于方向的表示，在图像旋转和翻转下性能下降更少，同时将边界IoU提升了高达3.7个百分点，从而实现了更精确的边界保持和物体定位。在ImageNet-1K分类、COCO目标检测和实例分割以及ADE20K语义分割上，VNCT始终优于方向扫描视觉SSM和一阶非因果SSM。这些结果表明，方向扫描对于高性能视觉SSM是不必要的，而二阶非因果状态空间建模为视觉识别提供了一种简单、高效且鲁棒的替代方案。

## Abstract
State Space Models (SSMs) have emerged as an alternative to Vision Transformers, yet most vision SSMs inherit directional token scanning from causal sequence modeling. While effective for sequential data, directional scanning introduces spatial bias and orientation-sensitive representations. We present Vision Non-Causal Trapezoidal Mamba (VNCT), a second-order non-causal vision SSM that enables all image tokens to interact in a single pass, eliminating direSctional scanning and achieving low single-image inference latency. VNCT exhibits more orientation-robust representations, showing reduced performance degradation under image rotations and flips, while improving Boundary IoU by up to 3.7 points, leading to more accurate boundary preservation and object localization. Across ImageNet-1K classification, COCO object detection and instance segmentation, and ADE20K semantic segmentation, VNCT consistently outperforms both directional-scanning vision SSMs and first-order non-causal SSMs. These results show that directional scanning is unnecessary for high-performance vision SSMs and that second-order non-causal state-space modeling offers a simple, efficient, and robust alternative for visual recognition.

## 1. 论文的核心问题与整体含义

- **研究动机**：现有视觉状态空间模型（SSMs）大多继承自因果序列建模，通过对图像令牌进行方向性扫描（如水平、垂直、多方向等）来序列化二维图像。这种扫描引入了**空间偏置**（spatial bias）和**方向敏感表示**，例如在图像旋转或翻转后性能显著下降。对于本质非因果的视觉场景（空间关系与令牌顺序无关），方向扫描是一种人为约束。
- **核心问题**：方向扫描是否是高性能视觉SSM的必要组件？能否彻底消除扫描，同时保持甚至提升性能？
- **整体含义**：本文证明方向扫描是不必要的，并提出了**二阶非因果状态空间模型 VNCT**，使所有图像令牌在单次前向传播中即可全局交互，无需任何扫描顺序。该方法不仅消除空间偏置，还提升边界保持能力与方向鲁棒性，且在多个视觉任务上超越现有扫描型和非因果一阶SSM。

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

- **核心思想**：将Mamba-3的二阶指数-梯形动力学（exponential-trapezoidal dynamics）提升为**非因果全局聚合算子**，替代传统的因果递归扫描。同时引入**低秩多输入多输出（MIMO）参数化**，联合建模空间与通道交互。
- **关键技术细节**：
  1. **NC-M3层（非因果Mamba-3）**：
     - 将原Mamba-3的因果递归 `h_t = α_t h_{t-1} + β_t x_{t-1} + γ_t x_t` 转化为**置换不变的非因果聚合**。
     - 定义交互核 `W_{ij,c}`，由查询 `ϕ(α_i)` 和源特征 `ψ(β_j + γ_j)` 构成，通过归一化得到注意力权重，但权重由SSM离散化参数（α, β, γ）而非可学习的QKV投影决定。
     - 利用**全局充分统计量** `S_c` 和 `Z_c` 实现线性复杂度 `O(LD)` 聚合：所有源令牌贡献求和一次，每个目标令牌通过内积读取全局状态，无需构造 `L×L` 矩阵。
  2. **二阶梯形离散化**：
     - 参数生成：`α = exp(A·Δ)`, `γ = σ(Λ)·Δ`, `β = (1-σ(Λ))·Δ·α`，其中 `σ` 是sigmoid，`Λ` 控制左右端点权重分配。
     - 相比一阶欧拉（ZOH）离散，梯形平均两个端点，带来更丰富的隐式卷积核。
  3. **低秩MIMO机制**：
     - 通过秩 `R` 的张量 `U` 将值流扩展到 `R` 个混合通道，全局状态 `G` 为三维张量，读回时再通过 `C` 投影，复杂度 `O(LCR)`，可减少全阶MIMO的参数量（典型减少24~192倍）。
  4. **2D旋转位置编码（RoPE）** 注入相对空间信息，保持置换不变性下的几何感知。
  5. **总体架构**：四阶段层次化骨干网络（图2），阶段1-3堆叠VNCT（NC-M3）块，阶段4用多头自注意力（MHSA）处理最低分辨率特征（开销可忽略）。每个VNCT块包含：局部感知单元（LPU，3×3深度可分离卷积）、2D正弦位置编码、NC-M3全局交互、层缩放残差、两层FFN。
- **算法流程**（以NC-M3块为例的文字描述）：
  - 输入 `X` 通过线性投影生成门控流 `z`、值流 `x`、B（源投影）、C（查询投影）、步长 `Δ`、梯形插值对数 `Λ`、RoPE角度 `Θ`。
  - 计算每个头部和令牌的 `α, β, γ`（公式12-14）。
  - 对B和C应用2D RoPE。
  - 对每个令牌计算融合源权重 `w_j = softmax(γ_j/√N) + softmax(β_{j+1}/√N)`，对应梯形左右端点。
  - 全局累加：`G = Σ_j w_j · x̃_j · B̃_j`（MIMO下增加秩维度）。
  - 每个目标令牌读出：`y_i = Σ_{r,n} G_{p,r,n} · C̃_{i,r,n}`。
  - 应用门控：`y = (h + D⊙x) ⊙ SiLU(z)`，再经输出投影还原通道数。

## 6. 论文的主要结论与发现

1. **方向扫描不是必要的**：VNCT在不使用任何扫描的情况下，在所有任务上超越扫描型SSM（VMamba等），证明方向扫描是多余约束。
2. **二阶非因果优于一阶非因果**：VNCT始终优于VSSD（一阶SSD），在ImageNet Top-1上高约0.5~0.6%，COCO AP高0.9，ADE20K mIoU高0.9，边界IoU高3.7。
3. **方向鲁棒性显著提升**：在翻转/旋转下的平均精度下降仅0.3%，远好于VSSD（0.9%）和VMamba（1.6%），表明表示更各向同性。
4. **边界保持能力更强**：边界IoU提升最多3.7个点，说明二阶非因果全局交互更好地保留了物体轮廓细节。
5. **推理延迟更低**：在Micro/Tiny/Small尺度下，VNCT延迟低于VSSD（如Micro: 5.25ms vs 5.80ms），因为消除了多扫描方向重复。
6. **有效感受野更优**：VNCT的ERF平滑、对称、覆盖全图，无扫描伪影（如VMamba的横条纹），也优于VSSD的不规则分布。
