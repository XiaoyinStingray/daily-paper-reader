## 摘要
大多数基础模型的自适应推理技术会改变模型的行为——提前退出、MoE路由、KV缓存压缩、动态注意力稀疏性。然而，输入到骨干网络的数据仍然是固定网格的分词，与图像内容无关。我们认为这是一个被忽视的杠杆。我们提出了SpiralFovea，一种无参数、输入自适应的分词器，其中词元的身份、位置、尺度和数量都是局部视觉熵的函数，且在任何骨干网络参数被查询之前完成选择。围绕内容驱动的热点锚点，多尺度螺旋环生成不超过78个图像块，替换输入阶段的196块标准ViT网格。在四个典型细粒度基准测试中，SpiralFovea在输入词元减少60%、每个Transformer层的自注意力FLOPs减少84%、吞吐量比匹配的静态分词基线提高18-29%的同时，准确率提升1.7-2.1个百分点。在CUB-200-2011 Genus上跨四个骨干网络的控制消融实验揭示了一个清晰的诊断：提升幅度与骨干网络全图位置先验的强度成反比，将自监督基础模型定位为输入自适应分词最有价值的领域。

## Abstract
Most adaptive-inference techniques for foundation models change what the model does - early exit, MoE routing, KV-cache compression, dynamic attention sparsity. The input that hits the backbone, however, remains a fixed-grid tokenisation indifferent to image content. We argue that this is a missed lever. We present SpiralFovea, a parameter-free, input-adaptive tokeniser in which token identity, location, scale, and count are all functions of local visual entropy and selection completes before any backbone parameter is queried. Around content-driven hotspot anchors, multi-scale spiral rings produce <= 78 patches that replace the standard 196-patch ViT grid at the input stage. Across four canonical fine-grained benchmarks, SpiralFovea yields +1.7-2.1 pp accuracy with a 60% reduction in input tokens, an 84% reduction in self-attention FLOPs at every transformer layer, and 18-29% throughput gains over the matched static tokenisation baseline. A controlled ablation on CUB-200-2011 Genus across four backbones reveals a clean diagnostic: the gain magnitude tracks inversely with the strength of the backbone's whole-image positional prior, isolating self-supervised foundation models as the regime where input-adaptive tokenisation is most valuable.

## 1. 核心问题与整体含义（研究动机和背景）

现有自适应推理技术主要围绕两个杠杆：一是改变模型架构/深度（如提前退出、MoE路由）；二是改变注意力模式（如稀疏注意力、KV缓存压缩）。然而，输入到骨干网络的**词元序列**始终是固定网格（如ViT的196个均匀patch），完全忽略图像内容的信息分布。这违反了率失真原则——最优容量分配应与局部信息密度成正比。论文提出**第三个杠杆**：在骨干网络参数被查询之前，根据图像内容自适应决定哪些词元存在、位于何处、尺度多大、数量多少，从而在推理阶段节省大量计算资源。

## 2. 方法论

### 核心思想
- **输入自适应词元化**：将词元选择视为图像内容的函数，而非固定分辨率。模仿人类视网膜中央凹（fovea）的机制：周边视觉进行显著性处理，将高密度感光细胞重定向到高信息区域。
- **参数无关**：所有选择过程无学习参数，基于局部视觉熵计算。

### 关键技术细节
1. **熵图计算**  
   - 将图像转为亮度（BT.601），下采样至D×D，量化至B个bin。  
   - 滑动窗口（半径r）内计算局部Shannon熵：  
     \[
     E(x,y)=-\sum_{b=0}^{B-1} \hat{p}_b(x,y)\log_2(\hat{p}_b(x,y)+\varepsilon)
     \]  
   - 再上采样回原分辨率。计算成本<单Transformer块的2%。

2. **热点锚点检测**  
   - 将图像垂直分为S条，每条第s条内取熵最大的位置（带边缘约束）作为锚点。  
   - 锚点归一化到[-1,1]²，并按半径τdedup去重，保证锚点是τdedup-填充集。

3. **多尺度螺旋环提取**  
   - 每个锚点周围放置K=4个同心环：环0是单个中心patch；环k>0有n_k个patch，角度均匀分布。  
   - 环的半径增长使各环无间隙拼接，patch大小随环递增（从24×24到48×48）。  
   - 每个锚点理想产生29个patch，4个锚点理论上限116个，实际因超出图像边界丢弃后通常≤78个。

4. **词元获取与嵌入**  
   - 通过双线性网格采样获取patch，超出边界阈值τoob=0.6的丢弃。  
   - 用骨干网络的冻结patch embed.proj嵌入，替换原始正弦位置编码为**极坐标位置编码**（2层MLP，输入锚点坐标、环索引、角度）。  
   - 拼接CLS词元，送入完整的冻结Transformer，最终CLS经线性头分类。

5. **CNN设置**  
   - 对于ResNet骨干，patch用共享ResNet-18主干编码，再用两个Mamba块融合变长序列（Mamba的O(N)循环处理顺序，对齐环序列）。

### 关键特性
- 选择在骨干网络前完成，无参数训练，不增加过拟合表面。
- 自注意力FLOPs降低约84%（(78/197)²），且每个层都受益，而非仅融合阶段。

## 6. 主要结论与发现

1. **严格帕累托改进**：在所有骨干和数据集上，SpiralFovea同时提升精度和效率（准确率+1.7~2.1pp，token减少60%，自注意力FLOPs降低84%，吞吐量提升18-29%）。  
2. **锚点放置主导增益**：熵引导的锚点比随机热点+1.5pp，多尺度环+1.1pp，螺旋布局+0.9pp，极坐标PE+0.5pp。  
3. **增益与位置先验负相关**：自监督DINO提升最大（+0.93pp），监督ViT-S/16接近零（-0.28pp，因饱和且强先验）。  
4. **第三杠杆的定位**：与架构级和注意力级自适应互补，可叠加形成乘法式帕累托改进。

## 7. 优点

- **创新性**：首次系统论证输入自适应词元化是独立于架构/注意力的第三杠杆。  
- **简洁高效**：无参数启发式，计算开销极小，直接插入冻结骨干。  
- **生物启发**：模拟视网膜中央凹工作机制，理论合理。  
- **严格诊断**：通过CUB-Genus跨骨干实验，给出可预测的增益边界条件，具实用性。  
- **开源透明**：报告种子标准差，详细超参数。

## 8. 不足与局限

- **锚点多样性有限**：仅通过垂直条带保证水平多样性，未使用学习式2D显著性检测器。  
- **变长序列批处理**：每张图像词元数不同，需填充，未实现桶式批处理。  
- **评估范围**：仅聚焦细粒度分类和医学成像（空间集中假设成立），未在ImageNet等全局分类上验证。  
- **边界案例**：监督ViT-S/16在CUB-Genus上退化-0.28pp，虽已解释（先验强+饱和），但提示在强监督先天先验下可能无效。  
- **缺失与现有AdaptFM技术的组合实验**：仅理论分析乘法式改进，未实证验证。
