## 摘要
我们研究了Transformer前馈模块架构设计的每个组成部分如何在初始化时决定秩随深度保留的程度。我们将长期以来被理解为控制量级的跳跃连接和归一化重新解释为在深度上保持梯度秩的机制，因为正是使网络具有表达能力的矩阵乘法和非线性激活也降低了秩。我们表明，跳跃连接在秩崩溃与集成行为之间进行权衡，这由分支和跳跃的相对尺度控制：跳跃连接将梯度绕过残差分支（秩在此处丢失），而不是沿着鼓励层组合的长梯度路径传播。归一化层的位置通过设置深度上的分支与跳跃比例来控制相同的权衡，统一了大部分关于归一化位置和深度缩放的研究，特别是为什么Post-Norm秩崩溃而Pre-Norm秩保持平稳。架构的其他方面，例如扩展和收缩宽度的双矩阵结构，使用额外的参数来保持表示或分支雅可比矩阵的秩。第二个矩阵去相关了一个连贯的平均尖峰，该尖峰在使用单一矩阵和无中心激活时会在各块间增长，防止了残差表示崩溃。两个矩阵之间的宽度扩展保持了分支雅可比矩阵的全秩：在此扩展空间中应用降秩激活保留了足够多的方向来覆盖原始空间，其宽度遵循Marchenko–Pastur定律。输入-输出雅可比矩阵的初始化秩预测了哪些网络能在CIFAR-10上训练。综合来看，我们将深度网络的架构设计重新诠释为在秩崩溃、集成行为和参数数量之间的内在权衡中导航。

## Abstract
We investigate how each component of the Transformer feedforward block architecture design determines how much rank survives across depth at initialization. We reinterpret skip connections and normalization, long understood as controlling magnitude, as mechanisms for preserving gradient rank across depth, since the very matrix multiplications and nonlinear activations that make the network expressive also reduce the rank. We show that skip connections trade off rank collapse against ensemble-like behavior, controlled by the relative scales of the branch and the skip: skip connections route the gradient around the residual branch, where rank is lost, rather than along the long gradient paths that encourage the layers to compose. The placement of the normalization layer controls this same tradeoff by setting the branch-to-skip ratio across depth, unifying much of the normalization placement and depth scaling literature, in particular why rank collapses for Post-Norm but plateaus for Pre-Norm. Other aspects of the architecture, like the two-matrix structure that expands and contracts the width, use additional parameters to preserve the representation or branch Jacobian rank. The second matrix decorrelates a coherent mean spike that would grow across blocks with a single matrix and uncentered activation, preventing the residual representation from collapsing. The width expansion between the two matrices keeps the branch Jacobian full rank: applying the rank-reducing activation in this expanded space leaves enough directions to span the original, at a width that follows a Marchenko--Pastur law. The initialization rank of the input--output Jacobian predicts which networks train on CIFAR-10. Taken together, we recast architecture design for deep networks as navigating an intrinsic tradeoff among rank collapse, ensemble-like behavior, and parameter count.

- **核心问题**：深度 Transformer 在训练中面临梯度秩塌陷（rank collapse）问题——输入-输出雅可比矩阵、残差表示和分支雅可比矩阵的有效秩随深度单调递减，导致信号退化、训练不稳定，并限制了深层网络的表达能力。同时，即使秩不塌陷，若梯度过多绕过分支（走跳跃连接），网络会退化为浅层集成（ensemble），丧失深层组合能力。
- **整体含义**：论文将深度网络架构设计重新诠释为**秩塌陷、集成行为与参数数量三者之间的内在权衡**。每个架构组件（跳跃连接、归一化位置、双矩阵结构、宽度扩展）都以不同方式影响这一权衡，理解这些机制可以指导更合理的架构设计，并解释为何 Pre-Norm 优于 Post-Norm、为何使用四倍宽度扩展等实践选择。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式

- **核心思想**：
  - 跳过连接和归一化通过**路由梯度绕过降秩的分支**来保留秩；分支的贡献由**分支-跳跃比例**$r_\ell = \|\beta J_\ell\|_F / \|I\|_F$ 控制。
  - 归一化位置决定该比例随深度的变化：Post-Norm 使比例恒定 → 秩塌陷；Pre-Norm 使比例随深度衰减（因为残差流尺度增长）→ 秩稳定。
  - 双矩阵结构（$W_{up}, W_{down}$）中的第二矩阵**去相关**不同层分支输出的均值，防止形成相干均值尖峰（coherent mean spike）导致表示秩塌陷。
  - 宽度扩展（expansion ratio $m/d$）在**扩展空间**中应用激活函数，使得降秩操作后仍保留足够多的方向来覆盖原始空间；分支雅可比矩阵的全秩阈值由 Marchenko–Pastur 定律给出：$m/d \ge 1/p(\sigma)$，其中 $p$ 是激活的存活率（ReLU 为 $1/2$）。

- **关键技术细节与公式**：
  - 残差块：$h_\ell = h_{\ell-1} + \beta f(h_{\ell-1})$，分支 $f(h) = W_{down}\,\sigma(W_{up}\,\mathrm{Norm}(h))$。
  - 输入-输出雅可比矩阵：$\frac{\partial h_L}{\partial h_0} = \prod_{\ell=1}^L (I + \beta J_\ell)$，其路径分解为 $\sum_{k=0}^L \beta^k S_k$，$S_k$ 为经过 $k$ 个分支的路径和，每条路径权重 $(\beta \alpha^M)^k$。
  - 分支-跳跃比例：$r_\ell = \|\beta J_\ell\|_F / \|I\|_F = \sqrt{p}\,\beta\alpha^M / \|h_\ell\|$（Pre-Norm 随 $\|h_\ell\|$ 增长而衰减）。
  - Post-Norm 下比例恒定，有效秩 $\mathrm{erank}$ 依赖于 $\beta\alpha^M\sqrt{L}$；Pre-Norm 下比例衰减，秩随深度稳定。
  - 第二矩阵去相关：单矩阵分支的层间均值余弦相似度为 $c_\ell = \mathbb{E}[\sigma]^2/\mathbb{E}[\sigma^2]$；双矩阵使其降至 0。
  - 宽度扩展：$J_f = W_{down} D W_{up}$，$D$ 为激活导数对角矩阵。存活行数 $n_{\mathrm{eff}} = p\,m$，Gram 矩阵 $G = \frac{1}{n_{\mathrm{eff}}}(D W_{up})^\top(D W_{up})$ 的特征值服从 Marchenko–Pastur 分布，当 $m/d \ge 1/p$ 时 $J_f$ 满秩。

### 6. 论文的主要结论与发现

1. **跳过连接恢复秩**：每经过一个前馈块，秩下降，跳过连接通过身份路径恢复秩，形成“锯齿”状模式。分支-跳跃比例 $\beta\alpha^M$ 控制滑动连接与集成行为的权衡。
2. **归一化位置决定分支-跳跃比例深度变化**：Post-Norm 使比例恒定，秩塌陷；Pre-Norm 使比例随深度衰减（残差流增长），秩保持平稳。投影项对秩损失贡献极小（<0.4%）。
3. **第二矩阵去相关均值尖峰**：单矩阵 + 无中心激活产生跨层对齐的均值，导致表示秩塌陷；第二矩阵随机旋转均值方向，消除对齐，层次均值的余弦相似度降至 0。
4. **宽度扩展保持分支雅可比满秩**：Activation 导数存活率 $p$ 决定了全秩阈值 $m/d = 1/p$。Marčenko–Pastur 定律准确描述谱，$m/d$ 越大条件越好。
5. **初始化有效秩预测可训练性**：CIFAR-10 上，当初始化有效秩低于约 0.2（归一化宽度）时，网络无法训练（准确率降为随机）。
6. **架构设计是三元权衡**：秩压缩、集成行为、参数数三者不可兼得——滑动连接和归一化牺牲组合换取秩，双矩阵和宽度扩展用参数换取秩。

### 7. 优点：方法或实验设计上的亮点

- **统一视角**：将多种架构组件（跳跃连接、归一化位置、分支结构、宽度扩展）纳入同一个“分支-跳跃比例”框架，解释了 Pre-Norm vs Post-Norm 等争议性实验现象。
- **理论推导结合实证**：推导了精确的 Marčenko–Pastur 阈值、交叉层均值对齐公式、缩放定律函数形式，并用 GPT-2 验证了训练后自由度的保留。
- **可预测性**：仅通过初始化时的雅可比有效秩即可判断网络是否能在 CIFAR-10 上训练，为诊断架构健康度提供简单指标。
- **消融精细**：巧妙分离了归一化的“缩放”与“投影”效应（脱钩缩放 vs 均值减除），明确归因于分支-跳跃比例而非投影。
