## 摘要
本文提出TECO，一种多维剪枝框架，通过协同剪枝卷积神经网络（CNN）的三个维度（深度、宽度和分辨率）来提高在嵌入式硬件上的执行效率。在TECO中，我们首先引入一个两阶段重要性评估框架，该框架根据每个维度内的局部重要性和跨维度的全局重要性高效且全面地评估每个剪枝单元。基于该评估框架，我们提出一种启发式剪枝算法，逐步剪枝CNN的三个维度，以在准确性和效率之间达到最优权衡。多个基准实验验证了TECO相对于现有最先进（SOTA）方法的优势。代码和预训练模型可在https://github.com/ntuliuteam/Teco获取。

## Abstract
In this paper, we propose TECO, a multi-dimensional pruning framework to collaboratively prune the three dimensions (depth, width, and resolution) of convolutional neural networks (CNNs) for better execution efficiency on embedded hardware. In TECO, we first introduce a two-stage importance evaluation framework, which efficiently and comprehensively evaluates each pruning unit according to both the local importance inside each dimension and the global importance across different dimensions. Based on the evaluation framework, we present a heuristic pruning algorithm to progressively prune the three dimensions of CNNs towards the optimal trade-off between accuracy and efficiency. Experiments on multiple benchmarks validate the advantages of TECO over existing state-of-the-art (SOTA) approaches. The code and pre-trained models are available at https://github.com/ntuliuteam/Teco.

## 1. 核心问题与整体含义（研究动机和背景）

卷积神经网络（CNN）在边缘设备（如自动驾驶、智能相机）部署时面临严重的计算与功耗约束。例如ResNet50有41亿次MAC运算，远超嵌入式硬件能力。现有剪枝方法大多只单一地剪枝一个维度（深度、宽度或分辨率），忽略了其他维度的冗余，导致精度与效率的次优权衡。本文旨在提出一种**多维度协同剪枝框架TECO**，同时优化CNN的深度、宽度和分辨率三个维度，以在嵌入式硬件上获得更好的精度-效率平衡。

## 2. 方法论

核心思想：通过两阶段重要性评估（局部重要性+全局重要性），在巨大设计空间中高效识别冗余单元，并逐步剪枝。

### 关键技术细节
- **两阶段评估框架**：
  - **INES（内维度评估策略）**：在每个维度内部快速评估单元的局部重要性。采用全门控残差块（fully gated residual bottleneck block），为每个通道添加通道门（CG），为每层添加层门（LG），通过一次反向传播即可获得所有门的梯度，将梯度平方作为局部重要性。对于分辨率维度，由于均匀缩小所有图像，不需内维度评估。
  - **ITES（跨维度评估策略）**：对每个维度中最不重要的单元（由INES选出），计算其全局重要性。全局重要性综合考虑三个因素：
    - 对精度的影响：移除单元后交叉熵损失的增加量 \(A(u)\)。
    - 对模型复杂度的影响：MACs减少量 \(M(u)\)。
    - 对推理延迟的影响：延迟减少量 \(T(u) = a_s \cdot M(u)\)（通过维度级延迟模型线性拟合得到）。
    - 综合指标：\(I(u) = \frac{A(u)}{\alpha M(u) + (1-\alpha)T(u)}\)，其中\(\alpha=0.5\)。
- **启发式剪枝算法**：迭代执行INES→ITES→剪枝→微调，逐步从三个维度中移除全局重要性最低的单元。

### 公式与算法流程（文字说明）
1. 给定过参数化CNN \(N\)，训练集 \(D_t\)，评估集 \(D_v\)（从训练集中随机选5000张图像）。
2. 对每个维度，执行INES得到局部重要性，选出各维度最不重要单元 \(d^*, w^*, r^*\)。
3. 执行ITES计算这三个单元的全局重要性，选择得分最低的单元进行剪枝。
4. 剪枝后微调1个epoch（学习率1e-4）。
5. 重复直到达到资源预算。

## 6. 主要结论与发现

1. TECO在ImageNet上实现了所有计算量级别下的最佳精度-效率权衡。例如TECO-S以1.05B MACs达到73.07% Top-1准确率，比DECORE-4高3.36%且MACs更少；TECO-M以2.24B MACs达75.68%，优于HRank及DR-ResNet50。
2. 在嵌入式硬件上，TECO显著降低推理延迟：在Jetson Nano上比GAL减少约50%延迟且精度更高；在AGX Xavier上比HRank加速1.91倍。
3. 在CIFAR-10上，TECO甚至比基准ResNet110精度更高（93.94% vs 93.50%），由于多维度剪枝缓解了过拟合。
4. 消融实验证明：同时使用INES和ITES可获得与单独使用ITES接近的精度，但剪枝时间降低83.21%（从179.42小时降至30.12小时）。

- **延迟模型**：提出线性延迟模型，可快速估计剪枝对延迟的影响，适应不同嵌入式平台。
- **全面实验**：覆盖多个数据集、三种硬件平台、多种对比方法，消融实验验证各组件贡献。
