## 摘要
压缩是智能的基础。能够将其训练数据表示为短码的模型已经发现了能够实现泛化的规律性。大型神经网络可能学习到远比其参数数量所暗示的简单得多的函数，但构建实现这种简单性的编码却极具挑战性。基于参数的方法（如量化）产生的码长随模型规模缩放，对参数存储的信息量不敏感。前置序列编码通过压缩训练轨迹绕过了这个问题，但无论模型学到多少，它都会对精确的数据序列进行编码，当数据具有高熵时会产生较大的码长。我们引入了重序列编码，其中教师模型从学生自身的分布中选择训练样本。学生的码仅记录这些选择，其比特代价仅发生在教师与学生不一致的地方。由此产生的码长与参数数量和数据熵无关，通常比前置序列编码短几个数量级，且优势随规模增长而扩大。这种压缩揭示了先前压缩器无法触及的现象。在固定损失的情况下，更大的模型和集成尽管参数更多，却被压缩到更小的尺寸。嵌入PAC-Bayes界后，重序列编码为数十亿参数的大语言模型提供了最先进的泛化保证，即使零误差也优于基于激进的后训练量化构建的界。在计算最优机制下，随着模型相对于数据集规模越来越可压缩，该界随规模增大而收紧。同样的编码预测，当训练多个epoch时，模型会逐渐过拟合。它还将数据集中可学习的信息与不可预测的随机内容分离，揭示出较低熵的文本比高熵的图像数据包含更多可学习的结构。

## Abstract
Compression is fundamental to intelligence. A model that can represent its training data as a short code has discovered regularities that enable generalization. Large neural networks may learn functions far simpler than their parameter counts suggest, but it is challenging to construct codes that realize this simplicity. Parameter-based methods such as quantization produce code lengths that scale with model size, insensitive to how much information the parameters store. Prequential coding bypasses this issue by compressing the training trajectory, but codes the exact data sequence regardless of how much the model learns, yielding large codes when the data has high entropy. We introduce requential coding, where a teacher model selects training samples drawn from the student's own distribution. The student's code records only these selections, which cost bits only where teacher and student disagree. The resulting code length is independent of parameter count and data entropy, and often orders of magnitude shorter than the prequential counterpart, with an advantage that grows with scale. This compression sheds light on phenomena inaccessible to prior compressors. Holding loss fixed, larger models and ensembles compress to much smaller sizes despite more parameters. Plugged into a PAC-Bayes bound, the requential code yields state-of-the-art generalization guarantees for billion-parameter LLMs, outperforming bounds built on aggressive post-training quantization even granted zero error. The bound tightens with scale in the compute-optimal regime, as models become increasingly compressible relative to dataset size. The same code predicts that models gradually overfit when trained for multiple epochs. It also isolates the learnable information in a dataset from its unpredictable, random content, revealing that lower-entropy text holds far more learnable structure than higher-entropy image data.

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：学生模型（被压缩的模型）训练数据由学生自身生成，教师模型从学生提出的样本中选择“值得训练”的样本，学生只需记录教师选择的样本索引（而非样本内容）。码长由师生分歧（KL散度）决定，与参数数量和数据熵无关。
- **关键技术细节**：
  - **重序列编码流程**：学生初始模型P₀，教师Q₀，共享伪随机种子。每一步t，学生从自身分布P_t生成候选样本序列Y^(0)_t, Y^(1)_t,...；教师Q_t通过相对熵编码（REC）接受一个索引i^*_t，使X_t = Y^(i^*_t)_t服从Q_t分布；学生只记录该索引的编码m_t（码长约KL(Q_t||P_t) bits），然后学生用X_t更新为P_{t+1}，教师用真实数据更新为Q_{t+1}。解码器仅需学生和消息即可重建整个训练轨迹。
  - **相对熵编码（REC）**：从参考分布P中采样候选，接受规则使接受的样本边缘分布为目标Q。期望码长约为KL(Q||P) + 2log(1+KL) + κ，κ<5.21。
  - **教师平滑**：使用教师参数的指数移动平均（EMA）作为REC目标，减少噪声。
  - **等损失投影**：周期性地将教师重置为学生状态，然后让教师在真实数据上短暂训练直至恢复原损失，降低后续师生KL。
- **公式说明**：累积条件期望码长≤ Σ_t [KL(Q_t||P_t) + 2log(1+KL(Q_t||P_t)) + κ]。大batch下对数项可忽略，近似为累积KL。

### 6. 论文的主要结论与发现

- **重序列编码压缩效率远超参数级和前置序列编码**：码长比前置序列短1-2个数量级，在相同损失下码长低于4-bit/参数。
- **更大模型更可压缩**：固定损失下，模型参数从百万级增长到十亿级，重序列编码码长反而下降（例如从约2×10^7 bits降到约1×10^7 bits）。
- **集成模型更可压缩**：8成员集成码长仅为单模型的约60%，尽管总参数更多。
- **泛化界随规模收紧**：在计算最优（D=20N）下，重序列编码的PAC-Bayes界击败理想无损4-bit PTQ界，且泛化间隙随模型增大而衰减为幂律（C ~ N^{-0.46}或N^{-0.15}），指向零间隙极限。
- **多epoch过拟合预测**：泛化界在1 epoch后开始变差，与模型逐渐记忆训练数据一致。
- **可学习信息分离**：重序列编码测得文本（FineWeb）可学习信息约10^8 bits，图像（CIFAR-5M）约10^7 bits，随机数据接近0，而前置序列编码因数据熵过大无法区分。

### 7. 优点

- **理论创新**：首次提出码长与参数数量和数据熵完全解耦的压缩框架，理论上界优于所有现有方法。
- **应用价值高**：直接用于PAC-Bayes界，得到实际可用的非退化泛化保证，突破了以往LLM泛化界仅靠量化或启发式的限制。
- **实验全面**：涵盖多种数据、规模、集成、多epoch场景，并进行了消融分析。
- **可解释性**：压缩码长直接量化了模型学习的信息量，能区分数据中的结构与噪声。

### 8. 不足与局限

- **计算开销**：编码器需要评估大量候选样本，实际传输时间可能远高于训练时间；作者指出编码时间随KL指数增长，解码时间接近普通训练。论文主要将其作为评估工具而非传输工具。
- **评估成本**：需要运行完整的师生训练流程来获得码长，无法像量化那样直接施加于已训练参数。
- **遗忘信息未利用**：重序列编码只增不减，早期学到的若后期遗忘仍会留在码中，可能冗余。
- **架构泛化性**：实验仅限GPT-2风格的自回归Transformer，未验证其他架构（如编码器-解码器、RNN、视觉Transformer）。

- **实验未覆盖超大规模**：最大模型约1B参数，未测试更大模型（如数十亿参数）的码长行为，但理论趋势外推合理。
- **缺少与其他非参数级压缩（如剪枝、低秩分解）的定量对比**，尽管这些方法在概念上不如重序列编码灵活。
