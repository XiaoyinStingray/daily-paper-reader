## 摘要
心理视觉模型表明，人类视觉通过首先形成中间抽象，将低级特征提取与高级认知解耦。相比之下，基于深度学习的视觉模型通常使用同质的空间层堆叠来提取和聚合特征，使得其决策过程不透明。在本文中，我们提出了深度视觉编码，这是一种受1990年代图像编码启发的学习频域表示，该编码量化了感知显著的频率，结合复数图像表示产生了心理视觉风格的抽象。该方法实现了首个基于心理视觉的深度学习框架，利用数据驱动的频谱滤波器学习在特定频率子带中编码任务相关语义结构。显著性分析表明，与常规卷积神经网络（CNN）产生的模糊区域相比，我们的心理视觉模型提取了高度可解释的物体部件。此外，我们发现我们的模型在模型扩展时对深度的依赖性低于CNN，因为我们的复数表示和学习抽象取代了深度空间层的作用。这些发现共同表明，心理视觉编码为更高效和透明的视觉模型提供了一条有前景的路径。

## Abstract
Psychovisual models suggest human vision decouples low-level feature extraction from higher cognition by first forming intermediate abstractions. In contrast, deep learning-based vision models routinely extract and aggregate features using homogeneous stacks of spatial layers, rendering their decision-making processes opaque. In this paper, we propose Deep Visual Coding, a learned frequency-domain representation inspired by 1990s image codes that quantised perceptually salient frequencies, which together with complex-valued image representations produces psychovisual-style abstractions. This approach enables the first psychovisual-based deep learning framework, utilizing data-driven spectral filters that learn to encode task-relevant semantic structures within distinct frequency sub-bands. Salience analyses reveal that our psychovisual models extract highly interpretable object parts compared to the amorphous regions produced by regular Convolutional Neural Networks (CNNs). Furthermore, we find that our models are less depth dependent than CNNs for model scaling, since our complex-valued representations and learned abstractions subsume the role of the deep spatial layers. Together, these findings demonstrate that psychovisual coding provides a promising path toward more efficient and transparent vision models.

### 论文提出的方法论

- **核心思想**：提出 **深度视觉编码（Deep Visual Coding, DVC）**，一种可学习的频域滤波模块，模仿心理视觉编码中的冠状子带设计，并在复数图像表示（由Phasor Blocks生成）上操作，从而产生心理视觉风格的抽象。整个框架称为 **PsychoNet**，明确模拟心理视觉处理流程。
- **关键技术细节**：
    1. **DVC 模块**：
        - **Spectral Branches**：将频谱分解为多个不相交的矩形子带（如[14,8]、[8,4]、[4,1]），通过DropCrop操作设定频率边界。
        - **Hadamard Block**：在每个子带上应用可学习元素级滤波器 \(X_i^{\text{filtered}} = \text{Softmax}(X_i \odot W_i)\)，模拟心理视觉编码的量化步骤，随后进行归一化和通道混合（1×1卷积）。
    2. **Phasor Blocks**：
        - 将实值空间特征转换为丰富的复数表示。Phasor (I) 执行初始实-复转换，Phasor (C) 进一步细化复数特征。核心是使用深度可分离卷积块生成虚部分量，打破共轭对称性，使学习到的滤波更特异。
    3. **PsychoNet 框架流程**：
        - 标准ResNet风格的ResBlocks → Phasor Blocks（生成复数表示） → 2D FFT（将特征从空间域转换到频域） → 频谱压扩（平衡DC与高频） → DVC（滤波并聚合子带） → 频域中的复数线性层输出分类结果。
- **算法流程**（文字说明）：输入图像 → 前几个ResBlocks提取低级空间特征 → Phasor Blocks转为复数特征 → FFT到频域 → 对频域应用压扩 → DVC通过多个Spectral Branches分解频带并应用可学习Hadamard滤波 → 各子带输出聚合后经全局平均池化和复数线性层得到分类logits。

### 论文主要结论与发现

1. **DVC 学习到稀疏的冠状频率模式**，与手工设计的心理视觉编码高度相似，表明这些模式可能是心理视觉抽象的自然表示。
2. **Phasor Blocks 专门提取语义上有意义的物体部分**（如狗耳朵、象牙、车轮），实部和虚部分量分别捕获不同尺度的特征。
3. **DVC 将物体部件按尺度分配到不同子带**，并在子带内进一步细分到不同通道，形成结构化抽象，直接用于决策。
4. **PsychoNet 显著降低对深度的依赖**：在匹配参数和FLOPs时，PsychoNet使用的层数远少于ResNet（如Eff-B比ResNet101少38%层数），性能却持平或更优。
5. **在多个基准上性能匹配或超过对应 CNN 基线**：PsychoNet 在 CIFAR-10/100、ImageNet-100 和 ImageNet-1K 上均优于ResNet同类模型，与ConvNeXt-S性能接近。
