## 摘要
散射变换实现了Lipschitz稳定性和平移不变性，但密集预测任务需要保留全局平均中丢失的空间结构。我们提出相位感知散射编解码器，通过在跳跃连接中显式保留相位信息来恢复该结构。在图像去噪（BSD68）任务中，打破平移不变性使PSNR提升+2.17 dB；相位保留额外增加+1.03 dB。一种新颖的空间洗牌消融实验（-1.26 dB惩罚）表明相位编码了位置依赖的结构。我们在第二个密集预测任务（ISIC皮肤病变分割）上进行了初步的可扩展性研究，完整的交叉验证正在进行中。这项工作推进了原则性的小波-深度学习集成，展示了在像素级预测中相位信息如何补充散射的稳定性-表现力权衡。

## Abstract
Scattering transforms achieve Lipschitz stability and translation invariance, but dense prediction tasks require preserving spatial structure lost in global averaging. We propose Phase-Aware Scattering Encoder-Decoder, which restores this information by explicitly preserving phase in skip connections. On image denoising (BSD68), breaking translation invariance improves PSNR by $+2.17$~dB; phase preservation adds $+1.03$~dB. A novel spatial shuffling ablation ($-1.26$~dB penalty) demonstrates phase encodes location-dependent structure. We conduct a preliminary extensibility study on a second dense prediction task (ISIC skin lesion segmentation), with full cross-validation as ongoing work. This work advances principled wavelet-deep learning integration, showing how phase information complements scattering's stability-expressiveness trade-off in pixel-level prediction.

# 基于相位感知小波散射编解码器的密集预测方法总结

## 1. 论文的核心问题与整体含义（研究动机和背景）
散射变换（Scattering Transform）通过级联小波模运算实现了Lipschitz稳定性和平移不变性，在分类任务中表现优异。然而，密集预测任务（如图像去噪、分割）需要保留像素级的空间结构信息，而标准散射变换中的全局平均和降采样（downsampling by 2<sup>J</sup>）会丢弃这些精细空间信息。本文的核心问题是：**如何在保留散射变换稳定性的同时，恢复密集预测所需的空间等变性与结构信息**？论文提出一种相位感知的散射编解码器，通过打破平移不变性（使用步长1处理）并在跳跃连接中显式保留相位信息，实现了空间结构恢复与数学可解释性的兼顾。

## 2. 论文提出的方法论：核心思想、关键技术细节
### 2.1 核心思想
- **解耦稳定性与不变性**：散射变换的Lipschitz稳定性来源于级联模运算，而平移不变性源于降采样和全局平均。本文保留级联模运算以保证稳定性，通过步长1处理（stride-1）和逐尺度平滑（per-scale smoothing）打破全局不变性，恢复空间等变性。
- **相位信息显式保留**：标准散射变换仅保留幅度信息，丢弃了复值小波系数的相位。本文在编码器-解码器跳跃连接中以极坐标形式（幅度 A<sub>j</sub>、相位 Φ<sub>j</sub>）显式传递相位，用于恢复边缘、奇异点等位置相关结构。

### 2.2 架构设计
- **编码器（固定散射变换）**：不学习参数，采用Kymatio库的复Morlet小波。执行步长1卷积，无降采样；逐尺度应用与尺度成比例的平滑窗口 φ<sub>j</sub>，保持全分辨率。提取复系数，然后计算幅度和相位：
  - A<sup>θ</sup><sub>j</sub> = |W<sup>θ</sup><sub>j</sub> x| + ε
  - Φ<sup>θ</sup><sub>j</sub> = atan2(Im(W<sup>θ</sup><sub>j</sub>), Re(W<sup>θ</sup><sub>j</sub>))
- **瓶颈层（可学习）**：单层3×3卷积，将高维散射系数（K<sub>in</sub>通道）压缩为C<sub>bn</sub>通道（如64）。
- **解码器（可学习）**：U-Net风格，逐级上采样。每级接收三个信息源：上采样的瓶颈特征、相位门控后的特征、以及来自编码器的极坐标（A<sub>ℓ</sub>, Φ<sub>ℓ</sub>）。包含门控网络（Ψ<sub>gate</sub>）和融合网络（Ψ<sub>fuse</sub>）：
  - 门控：将幅度和相位拼接，经两层卷积后接sigmoid生成空间注意力掩码，与上采样特征逐元素相乘。
  - 融合：将门控后的特征、幅度、相位拼接，经两层卷积（Conv-BN-ReLU）学习组合。

### 2.3 关键技术细节
- 极坐标表示用于跳跃连接，解耦相位和幅度梯度，利于反向传播。
- 编码器无学习参数，保持数学可解释性和稳定性保证。
- 解码器的门控机制设计为相位引导的注意力，但实验发现门控收敛为常数值，实际效果微弱。
