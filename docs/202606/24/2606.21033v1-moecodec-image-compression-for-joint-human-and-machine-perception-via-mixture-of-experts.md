## 摘要
面向机器的图像压缩需要一种统一的编解码器来服务于多个下游视觉任务。现有方法要么采用特定任务的端到端设计，导致参数和部署开销增加；要么依赖基于迁移的适应方法，这些方法仍然停留在外部附加和启发式任务设计层面。这两类工作共有的一个关键局限性在于其大致静态的计算模式，即对不同令牌应用相似的变换，而忽视了图像不同区域对机器感知具有显著不同的语义重要性和复杂度。我们提出MoECodec，一种支持单模型内多个下游任务的令牌感知图像压缩框架。MoECodec将基于Transformer的压缩模型中的FFN层替换为令牌级混合专家模型（MoE），使得计算能够根据输入内容和任务目标进行动态的令牌级调整。为了使MoE在压缩模型中有效工作，我们引入了稳定的路由策略，将专家选择路由与空间全变差正则化相结合，以鼓励空间上一致的分配；并提出了轻量级专家架构——分组洗牌MLP（GShMLP），以控制参数增长。大量实验表明，在传统图像重建和机器任务上，MoECodec均相较于基线方法取得了一致性改进。

## Abstract
Image compression for machines calls for a unified codec that serves multiple downstream vision tasks. Existing approaches either adopt task-specific end-to-end designs, raising parameter and deployment overhead, or rely on transfer-based adaptations that remain externally attached and heuristic task design. A key limitation shared by both lines of work is their largely static computation pattern, which applies similar transformations across tokens despite the fact that different image regions exhibit markedly different semantic importance and complexity for machine perception. We propose MoECodec, a token-aware image compression framework that supports multiple downstream tasks within a single model. MoECodec replaces the FFN layers in transformer-based compression model token-wise Mixture-of-Experts (MoE), enabling dynamic, token-level computation conditioned on the input content and task objective. To make MoE effective in compression model, we introduce a stable routing strategy that combines expert-choice routing with spatial total variation regularization to encourage spatially coherent assignments, and we propose a lightweight expert architecture, Group Shuffle MLP (GShMLP), to control parameter growth. Extensive experiments show consistent improvement against baselines on both conventional image reconstruction and machine tasks.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有图像压缩方法在服务于机器视觉任务（分类、检测、分割等）时存在两大局限：
  - **任务特定端到端设计**（如专用编码器-解码器对或多分支结构）导致参数冗余、部署成本高；
  - **基于迁移的适应方法**（如插入轻量适配器）仍依赖外部附加和启发式设计，无法在一个统一模型中内在地学习多任务适应能力。
  - 更根本的**共性局限**是**计算模式静态**：对图像所有令牌（token）施加相同的变换，忽略了不同区域对机器感知的语义重要性和复杂度差异。
- **研究动机**：迫切需要一种**统一、动态、令牌级**的压缩框架，能在一个模型中同时支持高质量图像重建和多种下游机器任务，根据输入内容和任务目标自适应分配计算资源。
- **整体含义**：将混合专家（Mixture-of-Experts, MoE）引入基于Transformer的端到端图像压缩，实现了细粒度的动态计算，为多任务联合优化提供了新范式。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 用**令牌级MoE层**替换Transformer模型中的标准FFN层，使每个令牌可根据自身内容和任务目标被动态路由到不同专家网络，实现**内容自适应、任务自适应的异构计算分配**。

### 关键技术细节
1. **MoE层结构**：
   - 每个MoE层包含一个路由网络和E个并行专家网络。路由网络根据中间特征 `X ∈ R^(B×S×d)` 计算令牌-专家亲和度，决定每个令牌由哪个（些）专家处理。

2. **路由策略：Expert-Choice (EC) Routing + 空间总变差正则化**：
   - **Expert-Choice路由**（[64]）：不同于传统Token-Choice（每个令牌独立选专家），EC由每个专家主动从完整令牌序列中**选择其最偏好的前k个令牌**。这天然保证了专家负载均衡，并提供全局上下文，缓解了基于点令牌化带来的空间碎片化路由问题。
   - **公式表示**：
     - 亲和矩阵：`A = softmax(X W_r)` （W_r ∈ R^(d×E)）
     - 每个专家e从前S个令牌中选出前k个：`I_(b,e) = TopK(A_(b,:,e), k)`，其中 `k = ⌊S·f_c/E⌋`
     - 门控张量G保留选中令牌的亲和度，其余置零。
     - 输出：`X_out_(b,s) = (∑_e G_(b,s,e) F_e(X_(b,s))) / (∑_e G_(b,s,e) + ε)`
   - **空间总变差正则化（Spatial Total Variance Regularization）**：
     - 将亲和度张量A重排为2D空间形式，施加各向异性TV正则：`L_r = (1/(BE)) ∑_b ∑_e (‖∇_h Ã_(b,e)‖_1 + ‖∇_w Ã_(b,e)‖_1)`，鼓励空间邻域专家分配平滑一致，减少盐-胡椒噪声。

3. **轻量级专家架构：Group Shuffle MLP (GShMLP)**：
   - 标准FFN有两个线性层（隐层4倍扩展），直接复制E份导致参数膨胀。
   - GShMLP采用**分组投影**：将通道分为G组，每组内执行独立的两层线性变换。两个分组投影之间插入**无参数通道混洗**（channel shuffle），恢复跨通道信息交互。
   - 参数量约等于标准FFN的**1/G**（当保持隐层大小不变时），有效控制MoE参数量与基线模型相近。

4. **训练目标**：整体损失 `L_total = L_rd + α L_r`，其中L_rd为率失真损失，α控制正则化强度（默认1e-3）。

## 6. 主要结论与发现

- **MoECodec显著提升率失真性能**：在重建任务上，三骨干的BD-Rate分别降低11.54%（TIC）、8.12%（TinyLIC）、6.42%（TCM），同时总参数减少4.5%~0.44%。
- **机器任务一致优于强基线**：以Full Finetune-TIC为anchor，MoECodec在分类、检测、实例分割上分别降低BD-Rate 7.74%、8.98%、9.72%，且激活参数量更少（6.25M vs 7.51M）。
- **技术要素有效**：
  - Expert-Choice路由优于Token-Choice，加上空间TV正则化进一步改善空间连贯性，降低BD-Rate从19.57降至15.44。
  - GShMLP在相似参数量下比缩减FFN（ratio=1）性能好得多（15.44 vs 19.07），接近密集FFN（14.82）但参数少42%。
  - 增加专家数量（E=8）可进一步提升性能（14.59）。
  - 编码器+解码器均使用MoE效果最佳。
- **计算分配可视化**：MoECodec实现了任务自适应计算——分类任务侧重语义目标，MSE侧重高频纹理；检测/分割对结构线索关注有微妙差异。

## 7. 优点

1. **首次将MoE引入图像压缩**，提出令牌级动态计算范式，实现单模型多任务联合，避免了外部附加模块。
2. **针对LIC特殊挑战的工程技术**：利用Expert-Choice路由解决稠密令牌化导致的路由碎片化，引入空间TV正则化增强空间一致性；设计GShMLP平衡性能与参数。
3. **广泛的实验验证**：覆盖重建和三类机器任务，三种骨干，大量消融；定量与定性结合有力支撑论点。
4. **参数效率高**：MoE层稀疏激活，总参数和激活参数均小于或等于基线，延迟反而更低（142ms vs 155ms for TIC）。
5. **方法简洁有效**：替换FFN为MoE，无需改动整体架构，易于在现有Transformer LIC上集成。
