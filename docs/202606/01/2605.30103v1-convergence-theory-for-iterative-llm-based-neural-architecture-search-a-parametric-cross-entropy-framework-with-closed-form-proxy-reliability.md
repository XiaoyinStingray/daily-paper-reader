## 摘要
大语言模型（LLM）越来越多地被用作迭代神经架构搜索（NAS）中的生成器，但此类算法尚无正式的收敛理论。我们将迭代LLM-NAS建模为可执行程序上的参数化交叉熵（CE）方法，并证明了六个结果：（1）在精英架构上迭代微调LLM等价于限制在LLM参数族内的CE更新；（2）跨周期的期望架构质量单调非减；（3）精英集概率以几何速率C_t >= 1-(1-ρ_0)^t收敛到不动点；（4）在一阶马尔可夫令牌错误模型下，基于delta的生成比全代码生成实现了严格更高的有效生成率；（5）MinHash-Jaccard新颖性过滤器防止模式崩溃；（6）代理可靠性具有闭式ρ_S = (6/π) arcsin(ρ_P(SNR)/2)，从而得出实用诊断条件σ^2_arch >> σ^2_noise，这是基于代理的可信排名的必要条件。在一个包含22个周期、三个LLM、六个数据集、3300个生成架构的实验中，测试结果定量验证了两个预测，两个预测在效应方向上得到证实，并解释了先前经验报道但未解释的代理可靠性上限效应。

## Abstract
Large language models (LLMs) are increasingly used as generators in iterative neural architecture search (NAS), yet no formal convergence theory exists for this class of algorithms. We model iterative LLM-NAS as a parametric Cross-Entropy (CE) method over executable programs and prove six results: (1) iterative LLM fine-tuning on elite architectures is equivalent to the CE update restricted to the LLM parametric family; (2) expected architecture quality is monotonically non-decreasing across cycles; (3) elite-set probability converges to a fixed point at a geometric rate C_t >= 1-(1-rho_0)^t; (4) delta-based generation achieves a strictly higher valid-generation rate than full-code generation under a first-order Markov token-error model; (5) the MinHash-Jaccard novelty filter prevents mode collapse; (6) proxy reliability admits the closed-form rho_S = (6/pi) arcsin(rho_P(SNR)/2), yielding the practical diagnostic sigma^2_arch >> sigma^2_noise as a necessary condition for trustworthy proxy-based rankings. Testing against a 22-cycle, three-LLM, six-dataset experiment with 3,300 generated architectures confirms two predictions quantitatively, two at direction-of-effect level, and explains the proxy-reliability ceiling effect previously reported empirically but left unexplained.

### 论文提出的方法论：核心思想、关键技术细节与算法流程

1. **核心思想**：  
   - 将迭代 LLM-NAS 视为一个**参数化 CE 方法**：在每一轮中，LLM 从当前参数 θ_t 定义的分布 p_θ 中采样架构，评估并筛选出精英集（质量高于阈值 τ 且通过新颖性过滤的架构），然后通过在精英集上最大似然估计（即最小化 KL 散度）来更新参数 θ_{t+1}。
   - 证明 **LLM 微调与 CE 更新在参数族限制下等价**（Theorem 1）。

2. **关键技术细节**：
   - **架构空间定义**：所有合法可执行的 `torch.nn.Module` 子类的 Python 程序集合；delta 生成则限定为对基线架构应用统一 diff 得到的子集。
   - **交叉熵更新形式**：  
     每轮参数更新为：θ_{t+1} = argmin_{θ∈Θ} KL(ˆp_{S_t} ∥ p_θ)，其中 ˆp_{S_t} 是精英集 S_t 的经验均匀分布。这等价于最小化交叉熵损失。
   - **收敛性结果**：
     - 期望质量单调非减（Theorem 2，在有限样本下需加入 O(N^{-1/2}) 的松弛项）。
     - 精英集概率 C_t = P_{a∼p_{θ_t}}(q(a) ≥ τ) 单调收敛到某个不动点 C*（Theorem 3）；若参数族足够表达（Assumption 1(d)），则 C* = 1 且几何速率下界为 C_t ≥ 1 - (1-ρ₀)^t（Corollary 4）。
     - 对于 LoRA 微调，由于表达力限制，只能收敛到 C* ≤ 1（实际观察为 73–76% 的平台）。
   - **Delta 生成优势**：采用**一阶马尔可夫令牌错误模型**（代替独立令牌假设），证明 delta 生成的有效生成率与全代码生成的有效生成率之比为 λ^{(α-1)L_full}·(π_δ/π_full)，由于 α<1，该比值 >1（Theorem 5）。校准后预测比值为 2.23，实验观测为 1.41（方向一致但数值偏低）。
   - **新颖性过滤防止模式崩溃**：MinHash-Jaccard 距离滤波器保持精英集 (1-τ_nov)-分离，从而保证生成分布不会退化为单一架构（Theorem 7）。
   - **代理可靠性理论**：假设架构的真值质量 q 与代理质量 ˆq 服从二元正态分布，则 Spearman 秩相关系数与 Pearson 相关系数间有闭式关系：ρ_S = (6/π) arcsin(ρ_P(SNR)/2)，其中 SNR = σ²_arch / σ²_noise，且 ρ_P = (1+SNR^{-1})^{-1/2}（Theorem 8）。由此导出实用诊断：要求 σ²_arch >> σ²_noise，才能信任代理排名。

3. **算法流程**（每轮迭代）：
   1. 从当前 LLM 参数 θ_t 采样 N 个架构候选。
   2. 对每个候选运行 1 轮快速代理评估，得到质量分数 q(a)。
   3. 筛选出质量 ≥ τ 且通过 MinHash-Jaccard 新颖性过滤的架构，形成精英集 A⁺_t。
   4. 更新语料库 S_t = S_{t-1} ∪ A⁺_t。
   5. 在 S_t ∪ L（静态语料，如 LEMUR 数据集）上对 LLM 进行 LoRA 微调，更新至 θ_{t+1}。

### 论文的主要结论与发现

1. **六个理论结果**全部证明：CE 等价性、质量单调性、精英概率收敛、delta 生成优势、新颖性过滤防崩溃、代理可靠性闭式公式。
2. **实验验证**：
   - **定量一致**：期望质量 Qt 呈单调非减趋势（平滑后从 ~0.62 升至 ~0.68）；新颖性过滤在全 22 轮中都持续有新架构被收录。
   - **方向一致**：精英概率 Ct 在 73–76% 平台（符合 LoRA 表达力天花板而非 C*=1）；delta 有效生成率比全代码高（0.713 vs 0.506，比值 1.41，方向正确但低于校准的 2.23）；DeepSeek 的 Spearman 不显著（p=0.102），与 SNR→0 导致的非显著预测一致。
   - **偏离校准值**：delta 比率 1.41 < 2.23（归因于一阶马尔可夫模型未捕捉长程相关，且 diff 格式无效降低了 π_δ）；Qwen 的 Spearman 0.635 远高于双变量正态预测的 0.10（解释为精度上限截断导致 Spearman 与 Pearson 解耦）。
3. **代理可靠性排序**：Mistral（SNR=0.330, ˆρ=0.926, p<10⁻⁶）> Qwen（SNR=0.011, ˆρ=0.635, p=0.011）> DeepSeek（SNR=0.011, ˆρ=0.495, n.s.），与理论预测的 SNR 单调性完全一致，且仅 Mistral 在 Bonferroni 校正后显著（p=7.7×10⁻⁷）。
4. **解释经验现象**：首次给出代理可靠性上限效应的解析原因——SNR 太低时，σ²_arch ≈ σ²_noise，导致相关系数很小。

### 不足与局限

- **马尔可夫模型缺陷**：一阶假设不能捕捉注意力机制导致的长程令牌相关性，导致 delta 比率的校准值（2.23）偏高。
- **LoRA 假设未验证**：Theorem 2 要求“质量单调投影假设”（Assumption 2），由于 LoRA 的低秩约束，该假设当前无法检验。因此 Theorem 2 应视为条件性结论。
- **实验覆盖有限**：所有实证数据来自一个现有 LLM-NAS 实验 [Adhikari et al., 2026]，缺乏跨 LLM、跨搜索空间（如非图像分类任务）的第三方验证。
- **样本量小**：代理可靠性实验每 LLM 仅 12–16 个完整训练对，导致统计不确定性较大（如 DeepSeek 观测 ρ=0.495 与预测 ~0.10 的偏差可能仅为抽样波动）。
- **代理可靠性假设**：假设 (q, ˆq) 服从二元正态分布；当 99% 精度上限截断时，Spearman 与 Pearson 耦合被破坏（Qwen 案例），扩展理论至截断支撑是一个未解决问题。
- **新颖性过滤的熵界**：保证 δ_t > 0 但 δ_t 随 |S_t| 增长衰减，并非一致下界；长时间运行后能否继续防止模式崩溃仍有待实践检验。
- **缺少超参数敏感性**：未分析阈值 τ、新颖性阈值 τ_nov、每轮样本数 N 等对收敛速率和最终质量的影响。
