## 摘要
我们提出了通道式向量量化（CVQ），一种新颖的图像标记化范式，用通道式标记替代了块式标记。与传统的为每个块特征向量分配离散标记的向量量化不同，CVQ对特征图的每个通道进行量化。这种表述将图像表示为视觉细节的离散层次，而不是空间块的网格。基于CVQ，我们引入了一种新的视觉自回归框架，采用“下一通道预测”。我们的通道式自回归（CAR）模型不是按光栅顺序逐块渲染图像，而是顺序预测图像通道，逐步生成更丰富的视觉细节。具体来说，它首先勾勒全局结构，然后细化细粒度属性，类似于人类艺术家的创作流程。实验表明：（1）CVQ在无需任何额外技巧的情况下，实现了16K+码本大小的100%码本利用率，并显著提高了重建质量；（2）CAR在DPG得分上达到86.7，在GenEval得分上达到0.79，展示了在文本到图像生成中的强大有效性。

## Abstract
We present Channel-wise Vector Quantization (CVQ), a novel image tokenization paradigm that replaces patch-wise tokens with channel-wise tokens. Unlike conventional vector quantization, which assigns a discrete token to each patch feature vector, CVQ quantizes each channel of the feature map. This formulation represents an image as discrete levels of visual details, rather than as a grid of spatial patches. Based on CVQ, we introduce a new visual autoregressive framework with "next-channel prediction". Instead of rendering images patch by patch in raster order, our Channel-wise Autoregressive (CAR) model predicts image channels sequentially, producing progressively enriched visual details. Specifically, it first sketches global structure and then refines fine-grained attributes, akin to a human artist's workflow. Empirically, we show that: (1) CVQ achieves 100% codebook utilization with a 16K+ codebook size without any bells and whistles, and substantially improves reconstruction quality over conventional VQ; and (2) CAR attains a DPG score of 86.7 and a GenEval score of 0.79, demonstrating strong effectiveness for text-to-image generation.

# 论文《Channel-wise Vector Quantization》详细中文总结

## 1. 核心问题与整体含义（研究动机与背景）
- **问题**：传统向量量化（VQ）对图像特征图进行**块级（patch-wise）**量化，即每个空间位置（1×1×c）分配一个离散索引。这种范式存在两大缺陷：
  1. **码本利用率极低**：因图像块间高度相似，大量码本向量成为“死码”，在大码本（如16384）下利用率仅4.5%，导致信息丢失和重建质量差。
  2. **不适合序列到序列建模**：图像本质是2D网格，强行展平成1D光栅顺序会破坏局部空间依赖，且难以施加自回归友好的排序（如嵌套dropout）。
- **动机**：借鉴人类绘画过程（先勾勒全局形状和颜色，再逐步添加细节），论文发现自编码器的不同通道编码了不同视觉信息（全局到局部）。因此，**改变量化轴从“空间”到“通道”**，可自然生成一个由粗到细的1D标记序列，同时解决码本利用率问题。

## 2. 方法论：核心思想、关键技术细节、算法流程
### 核心思想
- **通道级向量量化（CVQ）**：对特征图的每个通道（h×w×1）独立量化，每个通道视为一个全局特征向量，码本索引代表一种全局视觉模式。
- **通道级自回归（CAR）**：基于CVQ，将图像生成重新定义为**逐通道预测（Next-Channel Prediction）**，从全局结构逐步生成精细细节。

### 关键技术细节
1. **CVQ量化公式**：
   - 给定编码器输出 Z ∈ R^{h×w×c}，CVQ将其视为c个通道向量 z^{(k)} ∈ R^{h×w×1}（k=1..c）。
   - 量化：z_q^{(k)} = argmin_{e_n ∈ C_channel} ||z^{(k)} - e_n||_2 （使用Frobenius范数）。
   - 前向传播使用直通估计器（STE）。
2. **训练损失**：与VQGAN相同，包括ℓ2重建损失、承诺损失、LPIPS感知损失和PatchGAN对抗损失。
3. **嵌套通道丢弃（Nested Channel Dropout）**：
   - 在训练CVQ时，随机保留前c_keep个通道（c_keep ∈ {1,…,c}），其余通道置零，强制模型将最关键的视觉信息编码到靠前通道。
   - 使用自适应GAN权重 λ_GAN(c_keep) = λ₀ / (1+e^{-η(c_keep - c/2)}) 以稳定稀疏通道下的判别器训练。
   - 以概率α执行丢弃，否则使用完整通道训练。
4. **CAR生成框架**：
   - 图像表示为c个离散通道标记序列 X = {x^{(k)}}_{k=1}^c。
   - 训练解码器only Transformer（基于预训练LLM如Qwen3）以自回归预测下一个通道：p(X) = ∏_{k=1}^c p(x^{(k)} | x^{(<k)}, text)。
   - 使用两层MLP projector将通道嵌入投影到LLM隐藏维度。

## 6. 主要结论与发现
1. **CVQ在无需额外技巧下实现100%码本利用率**（码本大小16384、65536均接近100%），而传统VQ仅4.5%~1.1%。重建质量显著提升：256 tokens时rFID=2.60（VQ:4.84），1024 tokens时rFID=0.88（VQ:1.32）。

## 7. 优点（方法与实验设计亮点）
- **方法简洁有效**：仅改变量化轴，无需额外模块（如因子化、CLIP初始化）、复杂技巧或扩散解码器，便同时解决码本崩溃和1D序列建模问题。
- **生成过程可解释性强**：CAR逐步生成图像，从粗结构到细节，类似人类绘画流程。
- **兼容标准自回归范式**：直接复用LLM的next-token prediction设置，便于与语言模型统一。
- **重建质量全面超越主流VQ变体**：在同等token预算下，rFID、SSIM、PSNR均优于MoVQ、VQ-LC、SimVQ等。
- **消融实验设计清晰**：验证了嵌套dropout的必要性、码本规模的影响、渐进通道分析，结论可靠。
