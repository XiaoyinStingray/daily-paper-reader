## 摘要
近期关于神经纹理压缩的研究表明，可以学习到每个材质的小型纹理表示（由潜在纹理和小型多层感知器解码器组成），这些表示可在渲染过程中实时解码，以再现基于物理的着色模型输入。然而，现有方法需要对每个材质进行针对给定MLP和潜在配置的梯度下降优化。在这项工作中，我们训练一个单一的超网络，同时输出潜在特征和MLP的权重与偏置。尽管解空间是高维的，但该方法产生的质量与当前参考的神经纹理压缩器相当。我们还进一步扩展了该方法，以同时推断多个解码器，甚至生成能够学习超分辨率的解码器。

## Abstract
Recent work on neural texture compression has demonstrated that it is possible to learn small, per-material texture representations (composed of latent textures and a small Multi-Layer Perceptron decoder) that can be decoded in real-time during shading to reproduce the input to a physically based shading model. However, existing methods require performing gradient-descent optimization per material for a given MLP and latent configuration. In this work, we train a single hypernetwork that outputs both the latent features and the MLP's weights and biases. Though the solution space is high-dimensional, this approach produces results comparable in quality to the current reference neural texture compressors. We further extend this approach to infer multiple decoders at once or even produce decoders that learn super-resolution.

# 深度论文总结：使用超网络的神经纹理压缩

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：现代基于物理的渲染（PBR）需要多个纹理贴图（如漫反射、法线、粗糙度、金属度），这些贴图占用大量存储和带宽，尤其在实时应用（如游戏）中问题突出。硬件块压缩（如BCn）虽能缓解，但压缩比有限。近年来，神经纹理压缩方法（如NTC、BCF1）通过学习每材质的潜在纹理和小型MLP解码器，实现了更高压缩比，且支持实时解码。
- **关键瓶颈**：现有神经压缩方法需对每个材质单独进行梯度下降优化（如BCF1每材质约14分钟），计算成本高昂，不适合大规模材质库的离线处理（如游戏场景中数百个材质）。
- **核心目标**：训练一个单一的超网络（Hypernetwork），一次性输出所有材质的潜在特征和MLP权重/偏置，无需逐材质优化，同时保持与参考方法相当的压缩质量，并扩展支持多解码器推断和超分辨率。

## 2. 方法论：核心思想、关键技术细节与流程

### 核心思想
- 使用一个端到端训练的超网络 `net`，输入纹理集 `T`，直接输出：
  - 四个BC1格式的潜在纹理 `{lll0, lll1, lll2, lll3}`
  - 一个小型MLP（1个隐藏层，隐藏维度16/32/64）的全部权重和偏置 `wb`
- 超网络学习从输入纹理到压缩表示的映射，从而替代逐材质的梯度下降优化。

### 关键技术细节

#### 2.1 整体管线（图2）
```
输入纹理集 T → 超网络 net → (潜在纹理 lll_i, MLP权重 wb) → MLP解码 → 输出纹理 T~
```
- 训练损失：输入纹理 `T` 与解码纹理 `T~` 之间的误差，反向传播优化超网络参数 `φ`。

#### 2.2 超网络架构（图3）
- **整体结构**：基于Diffusion Transformer架构（DiT），结合交叉注意力。
- **三个主要模块**：
  1. **t-Encoder**：用卷积 + 交叉注意力将输入纹理集 `T` 编码为少量全局令牌 `ttt ∈ R^{N×D}`（N=16，D=512）。令牌数量需足够大以表示MLP权重（`|ttt| ≥ |wb|`）。
  2. **DiT Blocks**：将纹理集按4×4块展成图像令牌 `xxx ∈ R^{M×D}`（M=W×H/16），通过N_B个DiT块进行交叉注意力（图像令牌与全局令牌交互）。使用sandwiched normalization防止信号死亡。
  3. **BC1转换模块**：
     - 从全局令牌 `ttt` 通过线性层生成 `wb`。
     - 从图像令牌 `xxx` 通过卷积层生成每个潜在纹理的BC1块数据：两个RGB端点（e0, e1）和4×4混合参数α。对端点加偏移（-3和+3）再经sigmoid和量化，α经sigmoid和2位量化。支持VAR A和VAR B两种BCF1变体（不同分辨率输出）。

#### 2.3 训练设置
- **损失函数**：L1范数 + DISTS感知损失（随机通道排列），并加入随机抖动。
- **优化器**：Adam with weight decay，学习率5e-5。
- **数据集**：来自Polyhaven Textures（814材质集）、Polyhaven Assets（431材质集）、Sketchfab随机PBR资产（902材质集），大部分4096×4096分辨率。训练时从32×32到512×512随机裁剪，并应用随机颜色变换。

### 扩展任务（第4节）
- **共享MLP**：微调超网络，使所有材质共享同一组MLP权重（仅潜在纹理不同），避免运行时按tile分类MLP的开销。
- **多输出头**：单次推理同时输出多个不同隐藏维度（16/32/64）的MLP，提供不同质量/性能权衡。
- **超分辨率**：输入下采样纹理 `T_l`（mip 1），损失使用全分辨率纹理 `T_h`（mip 0），使超网络学习生成高分辨率细节。

## 6. 主要结论与发现

- 超网络可以成功学习将任意纹理集压缩为BCF1格式，且压缩质量与逐材质优化的BCF1方法相当（PSNR相近，但MS-SSIM略高，得益于感知损失）。
- 推理速度极快（2.4s/材质 vs 14min/材质），适合批量处理大规模材质库。
- 架构上Transformer优于U-Net和ConvNext；全局令牌数对性能影响不大（N=4到64结果稳定）。
- 共享MLP仅略降质量（<1dB PSNR, <0.02 MS-SSIM），但避免运行时分类开销。
- 超分辨率能力：输入低分辨率纹理可生成高分辨率细节（如木纹、刮痕），且无需配对的高低分辨率数据。

## 7. 优点：方法与实验设计亮点

- **核心创新**：首次将超网络应用于神经纹理压缩，消除逐材质优化，大幅降低离线处理时间。
- **高效架构**：采用DiT+交叉注意力，有效处理高维解空间；BC1转换模块设计巧妙（端点偏移、sigmoid、量化）避免压缩坍塌。
- **多任务泛化**：同一超网络可轻松扩展至共享MLP、多输出头、超分辨率，展现灵活性。
- **训练策略**：结合L1和感知损失（DISTS）、随机抖动、数据增强，提升泛化能力和感知质量。
- **公平对比**：与BCF1原方法使用相同压缩格式（BCF1）和相同硬件，控制变量公平。
