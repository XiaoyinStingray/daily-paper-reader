## 摘要
训练后量化（PTQ）能够利用少量数据实现深度网络的高效部署。然而，其在视觉自回归模型（VAR）中的应用仍相对未被探索。我们识别出将PTQ应用于VAR的两个关键挑战：（i）注意力值乘积中的较大重建误差，尤其是在粗粒度尺度上高注意力分数更频繁出现时；（ii）由于校准数据有限，码本条目的采样频率与其预测概率之间存在差异。为应对这些挑战，我们提出了一种针对VAR的PTQ框架。首先，我们引入了一种移位求和量化方法，通过聚合值标记的对称移位副本的量化结果来减少重建误差。其次，我们提出了一种用于校准数据的重采样策略，使码本条目的采样频率与其预测概率对齐。在类条件图像生成、修补、外推和类条件编辑上的实验表明，该方法在VAR架构上具有一致的改进，为VAR的PTQ建立了新的最先进水平。

## Abstract
Post-training quantization (PTQ) enables efficient deployment of deep networks using a small set of data. Its application to visual autoregressive models (VAR), however, remains relatively unexplored. We identify two key challenges for applying PTQ to VAR: (i) large reconstruction errors in attention-value products, especially at coarse scales where high attention scores occur more frequently; and (ii) a discrepancy between the sampling frequencies of codebook entries and their predicted probabilities due to limited calibration data. To address these challenges, we propose a PTQ framework tailored for VAR. First, we introduce a shift-and-sum quantization method that reduces reconstruction errors by aggregating quantized results from symmetrically shifted duplicates of value tokens. Second, we present a resampling strategy for calibration data that aligns sampling frequencies of codebook entries with their predicted probabilities. Experiments on class-conditional image generation, inpainting, outpainting, and class-conditional editing show consistent improvements across VAR architectures, establishing a new state of the art in PTQ for VAR.

# 论文总结：Shift-and-Sum Quantization for Visual Autoregressive Models

## 1. 核心问题与整体含义（研究动机和背景）

- **背景**：视觉自回归模型（VAR）通过“下一尺度预测”替代传统的“下一 token 预测”，在图像生成任务中表现优异。然而，VAR 包含多个 Transformer 块和迭代多尺度生成过程，计算资源需求大（内存、FLOPs）。网络量化（尤其是训练后量化 PTQ）是部署的关键技术，但 PTQ 在 VAR 上的应用尚不充分。
- **核心问题**：作者发现 VAR 量化面临两个独特挑战：
  1. **注意力-值乘积重建误差大**：尤其在粗尺度（低分辨率）上，由于 token 数量少，高注意力分数出现更频繁，导致量化误差放大。
  2. **码本条目采样频率与预测概率不匹配**：由于 PTQ 仅使用少量校准样本，随机采样导致码本条目的实际出现频率严重偏离其预测概率分布。
- **整体含义**：本文旨在设计针对 VAR 的 PTQ 框架，解决上述两大挑战，实现高效、低比特部署，同时保持生成质量。

## 2. 论文提出的方法论

### 核心思想
- **移位求和量化（Shift-and-Sum Quantization）**：通过对称复制并移位高注意力值 token，聚合其量化结果，从而降低注意力-值乘积的量化误差。
- **校准数据重采样（Calibration Data Resampling）**：重新分配校准数据中 token 的码本条目，使采样频率与预测概率分布对齐。

### 关键技术细节

#### （1）移位求和量化
- **动机**：粗尺度上高注意力 token 的量化误差方差大（公式 8 推导）。标准量化后，注意力分数和值 token 的噪声相乘导致误差大。
- **量化核（Kernel）**：对于值 `v`，定义 n 阶核：
  ```
  f_n(v; t_n) = (1/2n) * Σ_{k=-n}^{n-1} Q(v + (2k+1)*t_n)
  ```
  其中 `t_n = s/(4n)`，`s` 为量化步长。该核的量化误差上界为 `s/(4n)`（定理1）。
- **应用**：仅对“高注意力 token”（平均注意力分数 > 阈值 θ）应用该核。实际操作时：
  - 复制高注意力 token 及其对应注意力分数。
  - 对每个副本按 `± (2k+1)*t_n` 进行对称移位。
  - 量化后求和并除以 `2n`（通过位运算高效实现）。
- **自适应阶数**：根据 token 的注意力分数选择最小的 2 的幂次阶数，使缩放后的注意力分数低于 θ，满足 BOP（比特操作）约束。

#### （2）校准数据重采样
- **步骤**：
  1. 随机采样生成初始校准样本，计算每个码本条目 `k` 的**平均预测概率** `ˆp_k = (1/(N*T)) * Σ_i Σ_j p_k(i,j)`。
  2. 定义目标频率 `t_k = N * T * ˆp_k`。
  3. 识别过采样条目集 `O` 和欠采样条目集 `U`。
  4. 对属于 `O` 的 token，重新分配其码本条目：以归一化概率 `˜p_k(i,j)`（仅对 `U` 中的条目非零）进行重新采样，直到频率匹配。
- **效果**：使校准数据的码本分布更接近真实预测分布，提升量化参数校准的准确性。
