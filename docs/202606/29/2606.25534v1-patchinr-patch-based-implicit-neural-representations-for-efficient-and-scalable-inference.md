## 摘要
隐式神经表示(INR)为连续信号建模提供了一种有效方法，但经典的逐像素推理导致推理次数呈二次增长，在高分辨率应用场景中显著增加计算成本。为了解决这个问题，我们提出了一种基于补丁的方法，将非重叠补丁作为基本处理单元，并在单个前向传播中预测整个像素补丁，从而显著减少所需的推理查询次数。为了验证我们方法的有效性，我们在现场可编程门阵列(FPGA)平台上为INR模型提出了一种硬件加速架构，该架构具有可配置流水线并支持双精度计算。我们的基于补丁的INR实现了与像素级INR相当的重建质量（使用2×2补丁时PSNR为34.97 dB），同时仅以0.6%的参数开销将推理延迟降低了75%。

## Abstract
Implicit Neural Representation (INR) provides an effective approach for continuous signal modeling, but classical per-pixel inference results in quadratic growth in inference count, leading to dramatically increased computational costs in high-resolution application scenarios. To address this issue, we propose a patch-based approach that treats non-overlapping patches as fundamental processing units and predicts entire pixel patches in a single forward pass, significantly reducing the number of inference queries required. To validate the effectiveness of our approach, we propose a hardware acceleration architecture on the Field Programmable Gate Array (FPGA) platform for the INR model, which features a configurable pipeline and supports dual-precision computation. Our patch-based INR achieves comparable reconstruction quality to pixel-level INR (34.97 dB PSNR with 2 x 2 patches) while reducing inference latency by 75% with only 0.6% parameter overhead.

- **问题**：传统的隐式神经表示（INR）采用逐像素（per-pixel）查询模式，每个像素需要一次独立的前向传播。随着分辨率提升，推理次数呈二次增长（O(HW)），导致计算成本激增，难以满足高分辨率或实时应用需求。
- **背景**：INR 在连续信号建模（图像、3D 形状等）中具有分辨率独立和紧凑的优势，但逐像素策略与现代并行硬件（如 FPGA）的批量处理特性不匹配，造成冗余坐标获取和低效运算。
- **整体含义**：本文提出基于非重叠补丁（patch）的 INR 框架（PatchINR），将基本处理单元从单个像素变为大小为 P×P 的补丁，单次前向传播输出整个补丁的像素值，从而将推理次数降低为原来的 1/P²。同时设计配套的 FPGA 硬件加速器，实现高效可扩展的推理。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：重新定义 INR 的查询粒度。传统 INR 映射单个坐标 x 到像素值 y = fθ(x)。PatchINR 将图像划分为非重叠的 P×P 补丁，每个补丁的坐标 xp 作为输入，网络输出对应补丁的所有像素值 Yp ∈ R^{P×P×c}（公式 Yp = fθ(xp)）。此时推理次数从 HW 降至 N = HW / P²，计算复杂度从 O(HW·T) 降为 O(N·T)。
- **关键细节**：
  - 补丁大小 P 是超参数，控制推理减少与参数增加之间的平衡。
  - 网络结构仍保持连续、分辨率无关的特性，只是输出维度变大了。

## 6. 主要结论与发现

- PatchINR 在保持与逐像素 INR 相当的重建质量的同时，大幅减少推理次数和延迟。

- FPGA 加速器可以有效利用补丁级并行性，INT8 量化进一步降低资源消耗与延迟，适合资源受限的边缘部署。

## 7. 优点
