## 摘要
我们提出了GOOSE-M2F，这是Mask2Former针对ICRA 2026 GOOSE 2D细粒度语义分割（FGSS）挑战赛的任务特定适配。GOOSE基准测试涵盖64个细粒度类别，跨越非结构化户外地形，具有严重的长尾分布，其中稀有类别每张图像占据不到50个像素。我们在Swin-Large Mask2Former基线基础上进行了三项针对性贡献：（1）200个对象查询以消除表示饱和；（2）结合ASPP-lite和CBAM双注意力的特征精炼模块（FRM）；（3）辅助监督头，为稀有类别提供直接的逐像素梯度。多阶段训练策略结合了分布平衡损失、稀有类别复制粘贴增强、动态IoU感知重加权和指数移动平均（EMA）。在推理时，具有2D高斯核混合和4尺度测试时增强（TTA）的密集滑动窗口引擎带来了+10.57%的提升。GOOSE-M2F实现了70.08%的官方综合mIoU（细粒度63.55%，粗粒度76.61%），在GOOSE 2D FGSS排行榜上排名第三。代码和训练好的模型已在GitHub（https://github.com/Aditya-Lingam-9000/GOOSE-M2F）和Hugging Face（https://huggingface.co/XYZ9843/GOOSE-M2F）上公开。

## Abstract
We present GOOSE-M2F, a task-specific adaptation of Mask2Former for the GOOSE 2D Fine-Grained Semantic Segmentation (FGSS) Challenge at ICRA 2026. The GOOSE benchmark spans 64 fine-grained classes across unstructured outdoor terrain with a severely long-tailed distribution, where rare classes occupy fewer than 50 pixels per image. We extend the Swin-Large Mask2Former baseline with three targeted contributions: (1) 200 object queries to eliminate representational saturation; (2) a Feature Refinement Module (FRM) combining ASPP-lite and CBAM dual-attention; and (3) an Auxiliary Supervision Head that delivers direct per-pixel gradients for rare classes. A multi-stage training strategy pairs Distribution-Balanced loss, Rare-Class Copy-Paste augmentation, dynamic IoU-aware re-weighting, and EMA. At inference, a dense sliding-window engine with 2D Gaussian kernel blending and 4-scale TTA adds +10.57%. GOOSE-M2F achieves 70.08% Official Composite mIoU (63.55% fine, 76.61% coarse), placing 3rd on the GOOSE 2D FGSS leaderboard. Code and trained models are publicly available at GitHub: https://github.com/Aditya-Lingam-9000/GOOSE-M2F and Hugging Face: https://huggingface.co/XYZ9843/GOOSE-M2F.

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
基于Mask2Former（Swin-Large骨干，100查询）进行三项针对性贡献：扩展查询数量、特征精炼模块、辅助监督头，结合多阶段训练策略和推理增强。

### 关键技术细节

#### 架构概述
- 基础：mask2former-swin-large-cityscapes-semantic（HuggingFace预训练）。
- 前向流程：`ŷ = HungarianMatch(Dec_200(FRM(PixDec(f_{Swin}(x)))), y)`，其中：
  - `f_{Swin}`：Swin-Large骨干网络。
  - `PixDec`：6层MSDeformAttn像素解码器，输出B×256×H/4×W/4。
  - `FRM`：特征精炼模块。
  - `Dec_200`：200个查询的Transformer解码器。

#### 扩展对象查询（Expanded Object Queries）
- 默认Mask2Former使用100个查询（针对Cityscapes 19类）。GOOSE有64类且每场景15–20个语义区域，查询饱和导致分割冲突。
- 扩展到200个查询：保留原始100个嵌入，新增100个查询初始化为高斯扰动：`q_i = q_{i mod 100} + ε_i, ε_i ~ N(0, 0.02²I), i ≥ 100`。贡献+2–3%综合mIoU。

#### 特征精炼模块（FRM）
- 解决像素解码器单尺度H/4限制。
- **ASPP-lite**：并行空洞卷积（d∈{1,3,6,12}，64中间通道）加全局池化，通过1×1卷积融合：`F_fused = Conv1x1(Cat[F_d=1, F_d=3, F_d=6, F_d=12, F_global])`。不同扩张率分别针对微小结构（d=1）、中等目标（d=3,6）、大区域（d=12）、场景上下文（全局池化）。
- **CBAM**：级联通道注意力和空间注意力：
  - `F_ch = σ(MLP[AvgPool(F)+MaxPool(F)]) ⊙ F`
  - `F_sp = σ(Conv7x7[Cat(F, F_hat)]) ⊙ F_ch`
  - 保留残差连接 `x + F_fused`。
- 贡献+3–4%在植被和地形类别。

#### 辅助监督头（Auxiliary Supervision Head）
- 稀有类别（<50像素）在主要Mask2Former损失中从二分匹配中获得零梯度，因为其微小段被分配给更大、更低成本的竞争者。
- 辅助头独立于查询匹配，在H/4分辨率下应用直接像素级交叉熵：`L_aux = CE(Up(ŷ_aux), y; w_dyn)`, `L_total = L_M2F + 0.4 L_aux`。头部为两层Conv2d（256→256→64），推理时移除，零VRAM开销。
- 贡献+5–8%稀有类别。

#### 训练策略

- **分布平衡损失（DB Loss）**：为每个类别加权 `w_c = (1-β)/(1-β^{n_c})`，β=0.9999，稀有类别权重最高50×，剪切到[0.1,10.0]。
- **动态IoU感知重加权**：每个验证周期后，根据EMA模型的IoU更新每类损失权重：`w_c^{(dyn)} = f_tier(IoU_c)`，映射[0%,80%+]到乘数[5×,1×]。
- **稀有类别复制粘贴（RCCP）**：预提取稀有类别的裁剪，以概率0.85 alpha混合到训练图像中（软5×5高斯掩码）。微小对象（电线、管道、交通锥）额外缩放U(1.2,2.5)。
- **类别感知重复采样（CAS）**：根据最稀有类别比例过采样：`r_c = min(p_t/f_c, 3.0)`，t=0.06，稀有类别图像最多过采样3×。
- **EMA**：影子权重 `θ_EMA^{(t)} = 0.9995 θ_EMA^{(t-1)} + 0.0005 θ^{(t)}`，用于验证和最终提交，比原始权重高1.0–1.5%。
- **多阶段调度**：8个阶段，AdamW（权重衰减0.03），多项式LR衰减，1000步线性预热，梯度剪裁5.0。骨干Swin-Large学习率是解码器的1/5。
  - 预热阶段（阶段1-3）：骨干1e-6，解码器5e-6，稳定在55.6%。
  - 加速阶段（阶段4）：LR提升10×（骨干1e-5，解码器5e-5），突破平台期至56.4%；阶段5-7巩固至59.5%。
  - 细化阶段（阶段8）：骨干5e-6，解码器2.5e-5，退火到最终。
  - 其他设置：裁剪576×1152，有效batch 16（1样本×16梯度累积），标签平滑0.10，FP16混合精度。

#### 推理引擎
- 最大单项增益+10.57%来自推理策略，而非架构。
- **密集滑动窗口**：图像分块为896×896重叠裁剪，步长384（57%重叠），每个像素被多个裁剪覆盖（+4–5%）。
- **2D高斯核混合**：每个裁剪的logit图乘以2D高斯核，中心权重高，边缘权重低，消除接缝伪影。`k(x,y)=exp(-((x-c_x)²/(2σ_x²)+(y-c_y)²/(2σ_y²)))`，σ=0.5c。
- **4尺度×2翻转TTA**：每个图像以0.5×,0.75×,1.0×,1.5×尺度处理，带有和无水平翻转，共8视图，logit平均池化后argmax。1.5×放大对微小类别特别有效。使用EMA权重。总TTA提升：59.51%→70.08%（+10.57%）。
